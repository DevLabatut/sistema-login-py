# Sistema de Login Simples

Projeto simples de **sistema de login em terminal** desenvolvido em **Python**. O programa permite realizar cadastro e login utilizando e-mail e senha digitados pelo usuário.

## Preview

Execução ocorre diretamente no terminal/console.

## Funcionalidades

* Cadastro de usuário (e-mail e senha).
* Login com verificação de credenciais.
* Validação de campos obrigatórios.
* Área restrita após login bem‑sucedido.
* Senha oculta durante digitação.

## Tecnologias

* Python
* Biblioteca padrão `getpass` (oculta senha no terminal)

## Estrutura do Projeto

```
sistema-login-py/
│
└─ login_simples.py
```

## Como usar

1. Clone o repositório:

```bash
git clone https://github.com/DevLabatut/sistema-login-py.git
```

2. Entre na pasta do projeto:

```bash
cd sistema-login-py
```

3. Execute o programa:

```bash
python login_simples.py
```

4. Primeiro será feito o cadastro.

5. Depois realize o login usando os mesmos dados.

## Observações

* Os dados não são salvos em banco de dados; ficam apenas durante a execução do programa.
* Projeto voltado para aprendizado e prática de lógica em Python.
